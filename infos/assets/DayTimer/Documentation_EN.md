# Documentation of DayTimer

> This Asset is used to create procedual Skybox with Day/Night Cycle and Time.
> To start with this Asset is realy simple:

1. Change in the Lightning Settings the Skybox Material and define your Directional Light (Sun Source)
2. Create an empty GameObject and assign the DayTime Manager
3. Assign to the DayTime Manager the Material and Directional Light and manage your Sun, Moon and Stars Settings.
4. Finish, you can run your Game and have an functional Day/Night Cycle
